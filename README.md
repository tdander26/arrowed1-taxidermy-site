## Tech

Plain HTML, CSS, and vanilla JavaScript. No build step required — open `index.html` in any browser or deploy directly to GitHub Pages.

## Live Site

Hosted on GitHub Pages at: https://tdander26.github.io/arrowed1-taxidermy-site/

## Adding Real Photos

Drop images into the `/images/` folder and update the `style="background-image: url(...)"` attributes in `index.html` for the hero, gallery, and about sections.
